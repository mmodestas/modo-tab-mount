# PCB & Documentation

The project includes a custom PCB for power conversion and optional ESP32 integration.

## Specs
- Input: 110/230V AC  
- Output: 5V / 2A via USB-C  
- ESP32-C3 for smart switching  
- Optional WS2812 LED indicators
